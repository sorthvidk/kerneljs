View

src/view.js

$0 is the standard sorthvid content container class. All parameters are wrapped in ES6 object syntax.

new View(el: Element, content: String, events: Object, displayName: String, data: Object, mount: String)
Parameters
el (Element) The associated DOMelement OR a string shorthand for generating an element
content (String) Optional string HTML content to be injected into a generated element
events (Object) A json object containing the events for the instance
displayName (String) A huma readable name for the View
data (Object) A json object containing text strings to be added to markup with data attributes data-text=key
mount (String) if a mount is provided the view automatically mount to the given point
